# What is the ForCS Extension?

Forest Carbon succession simulates changes in the biomass of each cohort dependent upon age, competition, and disturbance. 

# Release Notes

- Latest official release: Version 4.1 — April 2026
- Full release details found in the User Guide and on GitHub.

# Requirements

To use ForCS Succession, you need:

- The [LANDIS-II model v8.0](http://www.landis-ii.org/install) installed on your computer.
- Example files (see below)

# Download and Install the Extension

- The latest version can be [downloaded from GitHub](https://github.com/LANDIS-II-Foundation/Extension-ForCS-Succession/blob/master/deploy/installer/LANDIS-II-V8%20ForC%20Succession%204.1-setup.exe).
- (Look for the download icon in the upper right.) 
- Next, launch the installer.
